
## disinstalla Angular 

`npm uninstall -g @angular/cli`

`npm cache clean`

`npm cache verify`

## installa Angular 
`npm install -g @angular/cli@latest`

## creo un workspace
`mkdir workspace_angular`

## creo un nuovo progetto Angular
`ng new HelloAngular`

## faccio partire il server per Angular
`ng serve --open`
